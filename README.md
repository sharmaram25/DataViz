## 🛠️ **For Developers: Quick Setup**

### **🔥 One-Line Install**
```bash
git clone https://github.com/sharmaram25/DataViz.git && cd DataViz && pip install -r requirements.txt && streamlit run app.py
```

### **🐳 Docker Deployment**
```bash
docker run -p 8501:8501 ramsharma/dataviz:latest
```

### **⚙️ Manual Setup**
```bash
# Clone and setup
git clone https://github.com/sharmaram25/DataViz.git
cd DataViz

# Create virtual environment
python -m venv dataviz_env
source dataviz_env/bin/activate  # Linux/Mac
# dataviz_env\Scripts\activate  # Windows

# Install dependencies
pip install -r requirements.txt

# Run the application
streamlit run app.py
```

### **🛠️ Troubleshooting Quick Fixes**

<details>
<summary><strong>🔧 Installation Issues</strong></summary>

```bash
# Clear cache and reinstall
pip cache purge
pip install --no-cache-dir -r requirements.txt

# Alternative with conda
conda create -n dataviz python=3.9
conda activate dataviz
pip install -r requirements.txt
```
</details>

<details>
<summary><strong>📊 Large File Handling</strong></summary>

```python
# For files > 200MB, use sampling
import pandas as pd
df_sample = pd.read_csv('large_file.csv').sample(n=10000)
# Then upload the sample to DataViz
```
</details>

<details>
<summary><strong>🚀 Performance Optimization</strong></summary>
